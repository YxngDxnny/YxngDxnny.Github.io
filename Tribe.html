Web Application for Tribe Virtual Styling

Client Name

Ikenna "Ayke" Chukwuemeka

Project Overview

Tribe is a virtual styling brand that creates virtual fashion style look book's and fashion styling reccomendations for any event, season or trend.

The main objective of this project was to develope a user-friendly, responsive, interactive and visually appealing web application, The application should enable users to input style preferences and other details like the type event/occasion, personal information, season, trend, colours etc. into the stylr quiz, users should also be able to check out different orders (style quizes) and make online card payments. when payments are confirmed the style book and reccomendation are sent via email by the administrator.

Key features include:

  • Storing orders (style quizes) in the database based on sessions rather than registration.
  • Cart System
  • Check out and payment system(Braintree payment portal)

Key Features

Shopping Cart

The Shopping Cart system takes style quizes as orders for the customer based on the current session It also enables users to edit the details of orders and delete orders from the cart.

Check out and payment system

The web application implements Braintree payment portal to process bank card payments
