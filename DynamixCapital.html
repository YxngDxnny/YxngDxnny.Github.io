Web Application for Dynamix Capital

Client Name

Dynamix Capital Investment LLC.

Project Overview

Dynamix Capital is a forward thinking independent, privately-owned wealth management firm offering family office, goals-based planning and investment management services to high-net-worth individuals and families, practice professionals, and business owners.

The main objective of this project was to develope a user-friendly, responsive, interactive and aesthetically pleasing web application, The web application should enable users to register, manage funds, manage their accounts and make investments in the different plans available from the user dashboard. Also, an admin dashboard to manage users and view activity on the site.

Other back-end features include:

  • An RSS feed with various news sources
  • A blog system in the admin dashboard to create posts for users
  • Automated emails to inform users and clients of activity.
  • 2-Factor Autentication System
  • User Referral System
  • On-site contact us form
  • Implementation of Google Maps, Translator and ReCaptcha v3

Key Features

User Registration

Users can register to the website to take advantage of the investment platform and fund management system, It allows users login with a password or with 2-factor authentication using popular authenticator applications.

Blog System

The blog system allows the website administrators create informative blog posts for the users of the website, it allows the administrator add a title, headings, paragraphs, line breaks and images.

RSS feed

The News page provides the breaking, latest and most informative news from the top news sources around the world like CNN, BBC, Fox etc.

2 Factor Authentication

The web application implements 2 factor authentication for an extra layer of security, users can use popular authenticator apps like Microsoft Authenticator, Apple Authenticator, Android Authenticator apps etc.
